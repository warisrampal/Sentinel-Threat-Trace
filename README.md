# Sentinel Threat Trace

## Educational Phishing Simulation & Detection Platform

### Overview

Sentinel Threat Trace is an educational cybersecurity project designed to demonstrate phishing simulation techniques while promoting cybersecurity awareness in a controlled environment.

This project is intended only for learning, demonstrations, and authorized security awareness activities.
